# Flutter Platform Date Picker

This provide default os date picker behaviour

# How to Use
```          
 PlatFormDatePicker(
      width: 200,
      onClick: (String formattedDate) {
        
      },
      prefixIcon: const Icon(
        Icons.date_range,
        color: Colors.black,
        size: 20,
      ),
      controller: controller,
      context: context,
    )
```
